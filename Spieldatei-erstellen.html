Hilfe
  • Geben Sie jedem Schüler eine Nummer,
    oder legen Sie eine Namensliste an.

  • Sie können nummerierte
    Trickot-Vorlagen herunterladen,
    oder die Blanko-Vorlagen
    mit Namen beschriften.

  • Sie können bereits vorhandene
    Namens- & Aufgabenlisten hochladen,
    und Einträge hinzufügen.

  • Sie können auch mehrere Aufgabenlisten
    nacheinander hochladen.

  • Mit "Datei speichern" erstellen Sie
    eine neue Spieldatei.
    Diese können Sie bei Spielstart verwenden.
